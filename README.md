# ETF_Analysis
## Project Objective
The purpose of this project is to analyse and screen ETF's for investment


### Methods Used
* Data Visualization
* Descriptive Statistics
* Web Scraping

### Libraries
* Pandas
* BeautifulSoup
* requests
* yahoo_historical

### Data
ETF Financial Metrics
List of Canadian ETFs
